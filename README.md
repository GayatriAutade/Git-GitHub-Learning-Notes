1. What is Git?
    Git is a version control system used to track changes in source code during software development.
   <br>
    Why Git is used?
        -popular Free and Opensource
        -Fast and Scalable
        -Also track history
        - Track changes in code
        - Work with multiple versions of a project
        - Collaborate with other developers
        - Restore previous versions if needed
      <br>
    Git Installation & Setup 
         - Official Git Website:[Download Git](https://git-scm.com/downloads)
         - Git Configuration Commands 
          Configure Username  - git config --global user.name "Your Name"
          Configure Email  - git config --global user.email "youremail@example.com"
          Check Configurations - git config --list
         -These commands are run in:
                - CMD
                - PowerShell
                - VS Code Terminal
                - Git Bash
          They set your Git identity for commits.
